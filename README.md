# Security Policy Library

A Structured collection of foundational security policies aligned with ISO 27001, NIST CSF, SOC 2, and GDPR.  
This library demonstrates governance maturity, documentation skills, and understanding of compliance frameworks.

## Included Policies
- Access Control Policy
- Data Classification Policy
- Acceptable Use Policy
- Vendor Risk Management Policy
- Incident Response Policy
- Password & Authentication Policy

## Structure
- /policies — finalized policies
- /templates — reusable templates
- /diagrams — optional visuals
